❖ Description 
Le but de ce projet est de réaliser un jeu du pendu fonctionnel et intéractif qui suit les règles suivantes : deviner un mot secret en proposant des lettres avec un maximum de 6 erreurs autorisées. A tout moment, l'utilisateur/le joueur peut proposer le mot complet s'il pense l'avoir deviné. 

◇ Technologie utilisée 
Pour mettre ce jeu du pendu en place, j'ai utilisé HTML, CSS et JavaScript. J'ai stylisé la base HTML avec le CSS et je l'ai rendu fonctionnel grâce à JavaScript.

❖ Les fonctonnalités principales 
    • CSS : grilles (dessin du pendu), flexbox (disposition des éléments), media queries (responsive design)
    • JavaScript : variable let, constante const, DOM (createElement, appendChild, classList), les événements (addEventListener, onclick)

◇ Les évolutions futures 
Créer un jeu du pendu avec différent niveau de difficulté : facile, intermédiaire, avancé. En fonction du niveau de difficulté, le joueur aura le droit à moins de faute autorisée. 
Possibilité de choisir un thème pour les mots à deviner.
